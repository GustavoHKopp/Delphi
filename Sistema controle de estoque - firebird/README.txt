Projeto criado com o intuito de aprender e aprimorar conhecimentos no delphi.

---Sobre as telas do sistema---
*Cadastro de produto: temos a função de incluir produtos novos, excluir caso não tenha nenhuma movimentação vinculada ao mesmo e trazendo o estoque do produto conforme as movimentacoes do mesmo.

*Gerenciamento de movimentacoes: temos a função de incluir uma nova movimentação e escolher o tipo(entrada ou saida) validando e impossibilitando de negativar o estoque, temos também a possibilidade de excluir as movimentações que não possuem itens associados. Após termos uma movimentação cadastrada conseguimos incluir os produtos a mesma.

* Editor SQL: nesta tela temos como realizar consultas e rodar comandos no banco de dados. Para verificar as tabelas que estão criadas no banco foi criado uma tabela chamada TABELAS. Clicando no botão F9 executamos a função de consultar validando se estamos realmente rodando um SELECT e bloqueando caso não seja, clicando F6 executamos a função de executar também validando se é um comando que precisa ser executado.

---Sobre o sistema---
*O sistema esta configurado para o formado MDI, carregando as telas somente quando abrimos a mesma, com validação para impossibilitar abrir uma tela que já esta criada na memoria e apenas trazendo para frente das demais.

*Possuimos a validação para não deixar fechar o sistema com alguma tela aberta.

*Possuimos tmbém um bloqueio ao tentar fechar alguma tela do sistema com registro em inclusão.

*O projeto foi iniciado uma criação em base MYSQL e o projeto foi alterado e finalizado em uma base Firebird.
